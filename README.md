# Fueling Calculator

Plan your race nutrition & track what you actually ate.

## Features

- Plan fuel per segment with carb, sodium and caffeine targets
- Log actuals after each event and compare to your plan
- Build a personal product library for faster planning
- Access your plans from any device
        
Check it out at [fueling-calculator.missfunmi.com](https://fueling-calculator.missfunmi.com/).
